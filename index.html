Welcome to Cancer Simulation home page!

Cancer Simulation is a highly-parametrized C# application for modeling cancerous tissue growth at cellular-level. This project was created as an honors capstone for UMass Amherst Computer Science

A Windows machine with Microsoft XNA Game Studio 4.0 and Microsoft Visual Studio Express installed is required to run this code.
